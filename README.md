# Overview

I wanted to learn how to make a simple Java program, so I came up with Zelda Adventure. It is a simple turn-based RPG where you create your hero, defeat enemies, go to certain locations, and collect weapons. 
This helped me understand how to make a simple, terminal-based game, and I can customize it to have different enemies, weapons, and locations.

[Zelda Adventure Demo Video](https://youtu.be/cnMo_c1AvQ8)

# Development Environment

* Visual Studio Code

* Java 25
